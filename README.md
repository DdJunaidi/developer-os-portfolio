# 🚀 Impactful Bento Portfolio

A high-performance, developer-centric portfolio featuring a **Bento Grid** layout and an interactive **Terminal Tile**. Built to showcase Full-Stack capabilities (React, Node, Python).

## ✨ Key Features
- **Modern Bento UI**: Built with CSS Grid and Tailwind v4.
- **Interactive Terminal**: Showcases backend logic through a simulated CLI.
- **Fully Responsive**: Adapts from 4-column desktop to mobile-first stack.
- **Fast Deployment**: Optimized for GitHub Pages via Vite.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS v4, Framer Motion
- **Icons**: Lucide-React
- **Environment**: Vite

## ⚙️ Customization
This project is designed to be easily forkable!
1. **Modify Projects**: Edit `src/data/projects.json` to add your work.
2. **Terminal Commands**: Update the `commands` array in `TerminalTile.jsx` to reflect your skills.
3. **Colors**: Change the Tailwind colors in `index.css` to match your brand.
